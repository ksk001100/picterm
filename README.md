# Picterm

TUI image viewer

![](screenshot.png)

## Install
```bash
$ cargo install picterm
```

or

```bash
$ git clone https://github.com/ksk001100/picterm
$ cd picterm
$ cargo install --path .
```

## Usage
```bash
$ picterm # => Current directory
$ picterm ./
$ picterm $HOME/Downloads/
```

## Support file format
- PNG
- JPG
- WebP
- BMP
- GIF
